# WireGuard Key Generator

Безопасный генератор ключей WireGuard с современным интерфейсом. Генерация происходит локально в браузере с использованием Web Crypto API.

## Особенности

- **Локальная генерация** — ключи генерируются в вашем браузере, никогда не передаются по сети
- **Curve25519** — использует криптографию на эллиптических кривых (RFC 7748)
- **256-бит безопасность** — криптографически стойкие ключи
- **Современный интерфейс** — светлая и тёмная темы с плавными анимациями
- **Удобное копирование** — один клик для копирования ключей
- **Сохранение в файл** — экспорт ключей в текстовые файлы

## Использование

Откройте приложение: https://mxwebber.github.io/wg_keys_generator/

## Функции

### Генерация ключей
1. Нажмите кнопку **"Generate New Pair"**
2. Приватный и публичный ключи будут сгенерированы мгновенно
3. Каждая генерация увеличивает счётчик пар ключей

### Работа с ключами
- **Copy** — копирует ключ в буфер обмена
- **Save** — сохраняет ключ в текстовый файл

### Темы
Переключайте между светлой и тёмной темами с помощью переключателя в правом верхнем углу.

## Технические детали

### Криптография
- Алгоритм: Curve25519 (Montgomery curve)
- Длина ключа: 256 бит
- Кодировка: Base64
- Clamping: Согласно спецификации WireGuard

### Реализация
- Чистый JavaScript (ES6+)
- Нативная реализация Curve25519 с использованием BigInt
- Web Crypto API для генерации случайных чисел
- Без внешних зависимостей

### Совместимость
- Современные браузеры с поддержкой:
  - BigInt
  - Web Crypto API
  - ES6+ модули
  - CSS Custom Properties

## Безопасность

✅ Ключи генерируются локально в браузере
✅ Приватные ключи никогда не передаются по сети
✅ Использует криптографически стойкий генератор случайных чисел
✅ Реализация соответствует RFC 7748
✅ Нет внешних зависимостей или CDN
